## 🚀 How to Run

### Prerequisites
- Java 11+ installed
- Maven installed

### Steps

```bash
# Clone the repository
git clone https://github.com/your-org/Software-Engineering-.git
cd Software-Engineering-

# Build the project
mvn clean install

# Run the project
mvn exec:java
```
